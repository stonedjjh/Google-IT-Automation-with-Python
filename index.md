## Certificado profesional: Serie de 6 cursos

Este certificado de nivel principiante, compuesto por seis cursos y desarrollado por Google, está diseñado para proporcionar a los profesionales de TI habilidades altamente demandadas —incluyendo Python, Git y automatización de TI— que pueden ayudarte a avanzar en tu carrera.

Saber cómo escribir código para resolver problemas y automatizar soluciones es una habilidad crucial para cualquier persona en el sector de TI. Python, en particular, es actualmente el lenguaje de programación más solicitado por los empleadores.

Este programa parte de tus fundamentos de TI para ayudarte a llevar tu carrera al siguiente nivel. Está diseñado para enseñarte a programar con Python y cómo utilizarlo para automatizar tareas comunes de administración de sistemas. También aprenderás a usar Git y GitHub, a localizar y depurar problemas complejos, y a aplicar la automatización a escala mediante la gestión de configuraciones y la Nube.

Este certificado se puede completar en aproximadamente 6 meses y está diseñado para prepararte para una variedad de roles en TI, como Especialista en Soporte de TI avanzado o puestos de Administrador de Sistemas Junior. Al finalizar, puedes compartir tu información con empleadores potenciales como Deloitte, Target, Verizon y, por supuesto, Google.

Recomendamos que tengas Python instalado en tu máquina. Para algunos cursos, necesitarás una computadora donde puedas instalar Git o pedirle a tu administrador que lo instale por ti.

Si deseas aprender los fundamentos del soporte de TI, consulta el Certificado Profesional de Soporte de TI de Google de nivel principiante.

Proyecto de aprendizaje aplicado

- Aprende a programar con Python sin necesidad de conocimientos previos de programación y utiliza Python para automatizar tareas comunes de administración de sistemas.

- Aprende a usar Git y GitHub para localizar y depurar problemas complejos.

- Aplica la automatización a escala mediante el uso de gestión de configuraciones y la Nube.

- Practica tus habilidades técnicas con proyectos prácticos, incluyendo un proyecto final (capstone) donde utilizarás tus nuevos conocimientos para resolver un problema de TI del mundo real.

## Cursos

- [Curso intensivo de Python](./Automatización%20de%20TI%20con%20Python/1-Curso-intensivo-de-Python/)

Este curso está diseñado para enseñarle los fundamentos para escribir programas sencillos en Python utilizando las estructuras más comunes. No es necesaria ninguna exposición previa a la programación. Al finalizar este curso, comprenderá las ventajas de la programación en las funciones informáticas; entenderá la sintaxis básica de Python; explorará diferentes editores de código y entornos de desarrollo; será capaz de escribir programas sencillos utilizando Python; averiguará cómo encajan los componentes básicos de la programación; y combinará todos estos conocimientos para resolver un problema de programación complejo.

Comenzaremos sumergiéndonos en los fundamentos de la escritura de un programa informático. Por el camino, obtendrá experiencia práctica con conceptos de programación a través de ejercicios interactivos y ejemplos del mundo real. Rápidamente empezará a ver cómo los ordenadores pueden realizar multitud de tareas: sólo hay que escribir el código que les diga lo que tienen que hacer.

- [Uso de Python para interactuar con el Sistema operativo](./Automatización%20de%20TI%20con%20Python/2-Uso-de-Python-SO/)

Este curso está diseñado para explorar cómo ejecutar Python localmente y organizar y utilizar código en diferentes archivos Python. Los alumnos aprenderán a leer y escribir diferentes tipos de archivos, a utilizar subprocesos y flujos de entrada, a crear scripts Bash y a utilizar expresiones regulares. Este curso también cubre las pruebas automáticas y explica cómo configurar un entorno de desarrollador en una máquina personal. Comenzaremos explorando cómo ejecutar Python localmente, y organizar y utilizar código a través de diferentes archivos Python. Luego aprenderemos a leer y escribir diferentes tipos de archivos, y a utilizar subprocesos y flujos de entrada. También nos sumergiremos en las secuencias de comandos Bash y las expresiones regulares, dos herramientas muy potentes para cualquiera que trabaje con sistemas. Incluso abordaremos las pruebas automáticas, que nos permiten automatizar la forma de comprobar si nuestro código es correcto. Para terminar, juntaremos todo esto utilizando las herramientas que hemos adquirido para procesar datos y generar informes automáticos. También explicaremos cómo configurar tu propio entorno de desarrollador en tu máquina. Este es un paso clave para poder escribir y desplegar potentes herramientas de automatización.
